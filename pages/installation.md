## Check you are setup

To check that your setup works, run the following line. If you get some output, then you are good to go!

```
$ bilby_pipe --version
bilby_pipe=0.3.12: (CLEAN) 4b6a4c3 2020-07-30 15:13:57 +1000 bilby=1.0.0:
release
```
